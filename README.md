# is

A micro library for naive type checking

## usage

```js
> is.string (someVar)

// or equivalently
> is (someVar, 'string')
```

## size

|						|				|
|-----------------------|---------------|
| Raw					| 1283 bytes	|
| Uglified				| 681 bytes		|
| Uglified + gzipped	| 313 bytes		|

## environment support

Any browser (as `window.is`), AMD, CommonJS, NodeJS
