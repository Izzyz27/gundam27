# izzy

A micro library for naive type checking

## usage

```js
> izzy.string (someVar)

// or equivalently
> izzy ('string', someVar)
```

## size

|						|				|
|-----------------------|---------------|
| Raw					| 1283 bytes	|
| Uglified				| 681 bytes		|
| Uglified + gzipped	| 313 bytes		|

## environment support

Any browser (as `window.izzy`), AMD, CommonJS, NodeJS
